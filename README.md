# 占星骰子记录与分析工具

一款功能完整的占星骰子应用，为占星爱好者提供骰子投掷、结果记录、解读分析和长期数据统计功能。通过精美的3D动画效果和丰富的知识库，帮助用户探索行星、星座、宫位组合的奥秘。

## ✨ 功能特性

### 🎲 核心占卜功能
- **3D骰子投掷**：逼真的3D骰子动画，支持5种动画预设（经典旋转、弹跳、螺旋、极速、魔法）
- **多力度投掷**：4种投掷力度（轻柔、标准、强劲、激烈），影响动画时长和强度
- **手势投掷**：支持向上拖拽手势投掷，根据拖拽速度自动识别力度
- **专注投掷**：内置呼吸引导动画，帮助用户静心凝神后再投掷
- **自动解读**：投掷后自动生成详细的占卜解读，包含行星落星座、行星落宫位、星座落宫位的完整分析
- **问题模板**：预设多种常见问题模板，快速选择问题类型

### 📚 知识库系统
- **行星知识库**：12颗行星的详细含义、关键词、领域、正负特质
- **星座知识库**：12个星座的元素、模式、守护星、详细解读
- **宫位知识库**：12个宫位的领域、关键词、正负特质
- **元素与模式**：火土风水四元素、开创固定变动三模式的完整说明
- **组合解读**：行星×星座、行星×宫位、星座×宫位的组合含义
- **搜索功能**：支持关键词搜索知识库内容

### 📝 记录管理
- **历史记录**：保存所有投掷记录，包含问题、类型、解读笔记
- **分类筛选**：按问题类型（爱情、事业、财运、健康、人际等）筛选
- **搜索功能**：支持按问题内容、笔记、行星、星座搜索记录
- **收藏集**：创建自定义收藏集，整理特定主题的记录
- **多维度排序**：支持按时间、行星、星座、宫位排序收藏集内的记录

### 📊 统计分析
- **频率统计**：行星、星座、宫位的出现频率排行
- **组合分析**：最常见的二元、三元组合及其出现次数
- **类型分布**：按问题类型分类展示各维度的分布情况
- **元素与模式分布**：火土风水四元素、开创固定变动三模式的分布
- **时间分布**：分析投掷的时间偏好、高峰时段
- **活跃度统计**：每日投掷次数、连续投掷天数、平均每日投掷
- **热力图**：行星×星座、行星×宫位、星座×宫位的三维热力图
- **分析报告**：自动生成个人占卜数据分析报告

### 🎨 个性化定制
- **骰子套装**：5套预设主题（经典占星、星云幻境、极光之夜、午夜幽蓝、黄金时代）
- **自定义套装**：支持创建个性化骰子套装，自定义颜色、动画、音效
- **音效系统**：6种音效预设，支持自定义投掷和停止音效
- **动画预设**：5种动画效果，可自定义时长和缓动函数

### 🌅 每日运势
- **每日签到**：每日生成专属运势，包含今日行星、星座、宫位
- **运势解读**：基于当日骰子组合的个性化关键词和建议
- **幸运指数**：幸运指数、幸运色、幸运数字
- **签到统计**：连续签到天数、最长签到记录、总签到次数

## 🛠️ 技术栈

- **前端框架**：React 18 + TypeScript
- **构建工具**：Vite 6
- **状态管理**：Zustand 5（含持久化中间件）
- **路由**：React Router DOM 7
- **样式**：TailwindCSS 3
- **图表**：Recharts 2
- **图标**：Lucide React
- **工具类**：clsx、tailwind-merge
- **数据存储**：localStorage（纯前端本地存储）

## 🎯 核心流程

1. **投掷骰子**：用户点击投掷按钮或向上拖拽
2. **生成结果**：系统随机生成行星、星座、宫位组合
3. **播放动画**：展示精美的3D骰子旋转动画
4. **自动解读**：根据结果生成详细的占卜解读
5. **记录保存**：用户输入问题、选择类型、添加笔记后保存
6. **数据分析**：在统计页面查看长期数据趋势和规律

## 🔧 核心功能实现要点

### 3D骰子动画
- 使用CSS 3D transforms构建立方体
- 6个面分别展示不同符号
- 支持多种动画预设和力度调节
- 动态光晕和阴影增强立体感

### 占卜解读系统
- 12×12行星落星座解读矩阵（144种）
- 12×12行星落宫位解读矩阵（144种）
- 12×12星座落宫位解读矩阵（144种）
- 智能综合解读生成算法

### 统计分析引擎
- 频率统计：各维度出现次数和百分比
- 组合分析：二元、三元组合的出现频率
- 热力图：三维度交叉分析可视化
- 时间分析：投掷习惯和偏好分析

## 📱 响应式设计

- **桌面端**：三栏布局，充分利用屏幕空间
- **平板端**：两栏布局，优化触控区域
- **移动端**：单列布局，简化复杂图表
- 所有交互元素确保最小48px触控区域
